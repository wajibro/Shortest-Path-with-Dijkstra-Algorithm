# 🗺️ Shortest Path with Dijkstra Algorithm - Central Surabaya Road Network

Aplikasi desktop interaktif untuk visualisasi graf rute kota Surabaya dan pencarian jalur terpendek menggunakan algoritma Dijkstra. Dibangun dengan Python, Tkinter, NetworkX, dan Matplotlib.

## ✨ Fitur

- 📊 **Visualisasi Graf Interaktif** - Menampilkan node (titik) dan edge (rute) dengan bobot jarak (km)
- 🔍 **Dijkstra Shortest Path** - Menghitung dan menampilkan jalur terpendek antara dua node
- 🎨 **Dark Theme UI** - Antarmuka modern dengan tema gelap yang nyaman di mata
- 💾 **Save as Image** - Menyimpan visualisasi graf sebagai file PNG
- 📋 **Detail Route Information** - Menampilkan langkah-langkah rute dan total jarak
- 🖱️ **Interactive Controls** - GUI lengkap dengan combobox untuk pemilihan titik awal dan tujuan

## 🛠️ Teknologi yang Digunakan

- **Python 3.14+**
- **Tkinter** - GUI framework
- **NetworkX** - Graf dan algoritma jalur terpendek
- **Matplotlib** - Visualisasi graf
- **Pandas** - Membaca data dari Excel
- **NumPy** - Manipulasi array dan perhitungan layout
- **OpenPyXL** - Engine pembaca file Excel
